# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, and engage with substance. Personal
attacks, harassment, or discriminatory language are not welcome and will be
moderated. Report issues to the repository maintainers via a private issue
or the GitHub abuse-reporting flow.
